<=====================================>

Touches :
- Q/D : Move
- E : Interact (Shaman, Cloche)
- Space :
-   Sol : Saute
-   Air : Double Saut (si Débloqué)
-   Mut : Sauter du mur (si Débloqué)
- Shift : Dash (si Débloqué)

Editeur de Niveau :

étape 1:
- Input :
-   Si laissé vide : Créer une nouvelle Tile

-   Si laissé vide : Modifie la tile sélecter
